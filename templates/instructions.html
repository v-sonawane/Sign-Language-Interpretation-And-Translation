Instructions

1. First of all you need to click the Start button at the bottom of the instructions.
2. After pressing the Start button it will take some time to load and then few windows will open.
3. There will be 3 windows one will capture the live camera feed and in the other two images you will be able to see the countour images.
4. In the window where you will be able to see the live camera feed there will be a green rectangle.

5. Now position your hand in the green box and make sure you have a clear background in the green rectabgle region.
6. Now to start capturing the hand sign and start transcribing it press 'C'.
7. The live text to speech will start automatically once it is loaded.
8. To quit the program press 'Q'.
